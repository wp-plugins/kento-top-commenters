Top Commentators list By Count Comments

== Description ==

Display top Commentators or contributer list by comment count.

Plugin Features

* Three Diffrent Style.
* Add Anywhere via widgets.
* Comments Count.
* Unlimited Commentators List. 



== Installation ==

Via WordPress -

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate please go widget page you will see widget "Kento: Top Commenters" 
4. Drag this to your sidebar, Set value for settings<br />
	<strong>Widget Title:</strong> <br />
	<strong>How Many Commenter Name to show:</strong><br />
	<strong>Select Your Style: </strong><br />
5. Hit save
